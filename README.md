# Tic-Tac-Toe

A classic Tic-Tac-Toe (Crosses and Noughts) game built with [Next.js](https://nextjs.org), React and Tailwind CSS.

## About

Play a local 3x3 Tic-Tac-Toe match against another player on the same device:

- The starting player (X or O) is chosen randomly at the start of each game.
- Click a cell to place your mark; turns alternate automatically.
- The winning line (row, column or diagonal) is highlighted with a strike-through line.
- If the board fills up with no winner, the result is a draw.
- Press **RESET** to start a new game — it pulses with a glow when the current game is over.
- A theme toggle in the top-right corner switches between light and dark mode.

## Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/Tim-Abozny/tic-tac-toe-ts.git
cd tic-tac-toe-ts
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to play.

## Scripts

- `npm run dev` — start the development server
- `npm run build` — build the production bundle
- `npm run start` — run the production build
- `npm run lint` — run ESLint
